## 🎯 功能概述

**解决的问题：** 用户对音箱说"播放周杰伦的晴天"时，指令由小爱云服务处理，不会经过你的本地服务。

**解决方案：** 通过持续监听音箱的对话记录，自动拦截播放指令，调用你的音乐服务获取 URL 并播放。

## ⚡ 快速开始

### 1. 配置

```bash
# .env 文件
ENABLE_CONVERSATION_POLLING=true
CONVERSATION_POLL_INTERVAL=2.0
MUSIC_API_BASE_URL=http://localhost:5050
```

### 2. 启动

```bash
cd backend
python3 -m uvicorn xiaoai_media.api.main:app --reload
```

### 3. 测试

对音箱说：
```
小爱同学，播放周杰伦的晴天
```
