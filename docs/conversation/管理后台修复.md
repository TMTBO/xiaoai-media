# 管理后台对话记录显示修复

## 问题

管理后台的对话记录页面无法正确显示对话内容。

## 原因分析

**数据格式不匹配：**

后端返回的字段：
```json
{
  "time": 1773883896585,
  "query": "播放周杰伦的稻香",
  "answer": ""
}
```

前端期望的字段：
```json
{
  "timestamp_ms": 1773883896585,
  "question": "播放周杰伦的稻香",
  "content": ""
}
```

## 修复内容

### 1. 后端接口修复

**文件：** `backend/src/xiaoai_media/api/routes/command.py`

**修改：** 在返回数据前转换字段名称

```python
@router.get("/conversation")
async def get_conversation(device_id: str | None = None):
    """Get latest conversation records from the speaker."""
    try:
        async with XiaoAiClient() as client:
            result = await client.get_latest_ask(device_id)
        
        # Transform to match frontend expected format
        conversations = [
            {
                "timestamp_ms": conv["time"],
                "question": conv["query"],
                "content": conv["answer"],
            }
            for conv in result
        ]
        
        return {"conversations": conversations}
    except Exception as e:
        logging.getLogger(__name__).error("Get conversation error: %s", e, exc_info=True)
        raise HTTPException(status_code=502, detail=str(e))
```

### 2. 前端显示优化

**文件：** `frontend/src/views/ConversationHistory.vue`

#### 改进 1：空回答提示

当小爱没有回答时，显示友好提示：

```vue
<div class="answer" v-if="conv.content">
  <!-- 有回答时显示 -->
</div>
<div class="answer" v-else>
  <div class="text" style="color: #909399; font-style: italic">
    暂无回答记录
  </div>
</div>
```

#### 改进 2：播放指令标记

自动识别并标记播放指令：

```vue
<el-tag v-if="isPlayCommand(conv.question)" type="success" size="small">
  <el-icon><MusicNote /></el-icon>
  播放指令（已拦截）
</el-tag>
```

识别逻辑：
```typescript
function isPlayCommand(query: string): boolean {
  const playPattern = /^(?:播放|打开)(?:歌曲)?(.+)$/
  const match = query.match(playPattern)
  if (!match) return false
  
  const content = match[1].trim()
  const controlKeywords = ['音量', '暂停', '继续', '停止', '下一首', '上一首']
  return content && !controlKeywords.some(kw => content.includes(kw))
}
```

#### 改进 3：自动刷新功能

添加自动刷新开关，每 3 秒自动更新对话记录：

```vue
<el-switch
  v-model="autoRefresh"
  active-text="自动刷新"
  @change="toggleAutoRefresh"
/>
```

```typescript
function toggleAutoRefresh(enabled: boolean) {
  if (enabled) {
    fetchConversations()
    refreshTimer = window.setInterval(() => {
      fetchConversations(true) // 静默刷新，不显示提示
    }, 3000)
  } else {
    if (refreshTimer) {
      clearInterval(refreshTimer)
    }
  }
}
```

## 字段映射

| 后端内部 | API 返回 | 前端使用 | 说明 |
|---------|---------|---------|------|
| time | timestamp_ms | timestamp_ms | 时间戳（毫秒） |
| query | question | question | 用户问题 |
| answer | content | content | 小爱回答 |

## 已知问题

### 问题：小爱回答为空

**现象：** `content` 字段经常为空字符串

**原因：** 
- 小爱的某些回答可能没有 TTS 文本
- 或者对话 API 返回的数据中没有 answers 字段

**影响：** 不影响功能，只是显示"暂无回答记录"

**未来优化：**
- 可以尝试从其他字段获取回答内容
- 或者显示更多对话元数据
