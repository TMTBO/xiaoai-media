# 播放模式快速入门

## 概述

播放控制器使用定时器模式，根据音频时长自动设置定时器，在歌曲结束前触发下一曲。

## 快速开始

### 1. 配置

播放控制器默认启用，无需额外配置。

### 2. 使用

1. 创建播放列表并添加音频文件
2. 点击"播放"按钮
3. 系统会自动在每首歌曲结束时播放下一曲

## 工作原理

定时器模式的工作流程：

1. 播放开始时获取音频时长
2. 根据时长设置定时器
3. 定时器触发时自动播放下一曲
4. 重复上述流程

## 常见问题

### Q: 如何查看播放状态？

A: 在前端播放器栏可以看到当前播放状态、进度和歌曲信息。

### Q: 如何停止自动播放？

A: 点击"停止"按钮即可停止播放。

### Q: 切换时机不准确怎么办？

A: 检查音频文件的时长信息是否准确。
