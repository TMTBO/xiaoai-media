# 功能特性详解

XiaoAI Media 的所有功能特性详细说明。

## 🎵 音乐播放

### 功能概述
支持多平台音乐搜索和播放，包括 QQ 音乐、网易云音乐等。

### 主要功能
- **音乐搜索** - 按歌曲名、歌手名搜索
- **排行榜播放** - 支持各平台热歌榜、新歌榜等
- **语音播放** - 通过语音命令播放音乐
- **播放控制** - 播放、暂停、停止、上一首、下一首

### 使用方式
- Web 界面：音乐搜索页面
- API 调用：`POST /api/music/play`
- 语音命令："播放周杰伦的歌"

详见：[音乐播放文档](playback/README.md)

---

## 📋 播放列表

### 功能概述
创建和管理自定义播放列表，支持语音命令快速播放。

### 主要功能
- **创建播放列表** - 支持多种类型（音乐、有声书、播客等）
- **管理播放项** - 添加、删除、排序播放项
- **语音关键词** - 设置语音关键词，通过语音播放
- **批量导入** - 从目录批量导入音频文件
- **自然排序** - 智能排序（第1章、第2章、第10章）

### 存储优化
- 多文件存储结构（v1.0+）
- 列表加载速度提升 80-90%
- 支持大量播单和播放项

### 使用方式
- Web 界面：播单管理页面
- API 调用：`POST /api/playlists`
- 语音命令："播放音乐播单"

详见：[播放列表文档](playlist/README.md) | [批量导入指南](playlist/README_BATCH_IMPORT.md)

## ⏰ 定时任务

### 功能概述
创建定时任务，在指定时间自动执行操作。

### 任务类型
- **定时播放音乐** - 在指定时间播放歌曲
- **定时播放播放列表** - 在指定时间播放播放列表
- **定时提醒** - 在指定时间语音提醒
- **定时执行指令** - 在指定时间执行任意语音命令

### 触发方式
- **Cron 表达式** - 周期性任务（每天、每周、每月）
- **具体时间** - 一次性任务
- **延迟执行** - 延迟指定分钟后执行

### 使用场景
- 每天早上 7 点播放起床音乐
- 工作日晚上 8 点播放放松音乐
- 每 2 小时提醒喝水
- 30 分钟后停止播放

### 使用方式
- Web 界面：定时任务页面
- API 调用：`POST /api/scheduler/tasks/cron`
- 快捷 API：`POST /api/scheduler/quick/reminder`

详见：[定时任务文档](scheduler/README.md)

## 🎤 语音命令

### 功能概述
执行自定义语音命令，支持智能解析和自动响应。

### 命令类型
- **播放控制** - 播放、暂停、停止、上一首、下一首
- **音量控制** - 音量加、音量减、设置音量
- **音乐播放** - 播放歌曲、播放排行榜
- **播放列表** - 播放自定义播放列表
- **其他命令** - 查询天气、设置闹钟等

### 智能解析
系统会自动识别：
- 播放列表关键词（"播放音乐播单"）
- 排行榜名称（"播放网易云热歌榜"）
- 音乐搜索（"播放周杰伦的歌"）

### 使用方式
- Web 界面：语音指令页面
- API 调用：`POST /api/command`
- 对话监听：自动拦截和处理

详见：[语音命令文档](../backend/src/xiaoai_media/services/README.md#voicecommandservice)

## 🎧 对话监听

### 功能概述
自动监听音箱的对话历史，拦截和处理播放指令。

### 主要功能
- **自动拦截** - 拦截包含唤醒词的播放指令
- **智能识别** - 识别播放列表、排行榜、音乐搜索
- **自动响应** - 执行自定义播放逻辑
- **对话记录** - 记录所有对话历史

### 工作流程
1. 监听音箱对话历史
2. 识别包含唤醒词的指令
3. 解析指令内容
4. 执行相应操作
5. 记录对话历史

### 配置选项
```python
# user_config.py
ENABLE_CONVERSATION_MONITOR = True
WAKE_WORDS = ["小爱", "播放"]
ENABLE_WAKE_WORD_FILTER = True
```

### 使用方式
- 自动运行：启动服务后自动监听
- Web 界面：对话记录页面查看历史
- API 调用：`GET /api/conversation/history`

详见：[对话监听文档](conversation/README.md)

## 💬 TTS 播报

### 功能概述
将文字转换为语音，通过音箱播报。

### 主要功能
- **文字转语音** - 支持中文和英文
- **设备选择** - 可选择任意设备播报
- **即时播报** - 实时转换和播放

### 使用场景
- 定时提醒（"该喝水了"）
- 语音通知（"快递已到达"）
- 智能家居联动（"温度过高"）

### 使用方式
- Web 界面：TTS 朗读页面
- API 调用：`POST /api/tts`
- 定时任务：结合定时任务自动播报

详见：[TTS 文档](tts/README.md)

## 📱 设备管理

### 功能概述
管理多个小爱音箱设备，查看设备状态。

### 主要功能
- **设备列表** - 查看所有设备
- **设备状态** - 在线/离线状态
- **播放状态** - 播放中/暂停/停止
- **设备信息** - 型号、IP 地址等
- **全局选择器** - 统一的设备选择入口

### 支持的设备
- 小米智能音箱 Pro (OH2P)
- 小米智能音箱 (LX06)
- 小爱音箱 Play (LX01)
- 小爱音箱 mini (LX04)
- Redmi 小爱音箱系列
- 其他小米生态链音箱

### 使用方式
- Web 界面：设备列表页面
- API 调用：`GET /api/devices`
- 全局选择器：页面顶部设备选择

## 🔌 REST API

### 功能概述
完整的 REST API，支持所有功能的编程访问。

### API 分类
- **设备管理** - `/api/devices`
- **音乐控制** - `/api/music/*`
- **播放列表** - `/api/playlists/*`
- **定时任务** - `/api/scheduler/*`
- **语音命令** - `/api/command`
- **TTS 播报** - `/api/tts`
- **音量控制** - `/api/volume`
- **对话历史** - `/api/conversation/*`
- **配置管理** - `/api/config`
- **实时状态** - `/api/state/stream` (SSE)

### 特性
- RESTful 设计
- JSON 格式
- 完整的错误处理
- SSE 实时推送
- 详细的 API 文档
