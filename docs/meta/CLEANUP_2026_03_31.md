## 使用指南

### 配置系统
1. 复制配置模板：
   ```bash
   cp user_config_template.py user_config.py
   ```

2. 编辑配置文件：
   ```bash
   vim user_config.py
   ```

3. 参考模板中的注释进行配置

### 运行测试

#### 单元测试
```bash
cd backend
pytest tests/
```

#### 功能测试
```bash
# 认证测试
python test/test_auth.py

# 对话测试
cd test/conversation && ./test_conversation.sh
```
