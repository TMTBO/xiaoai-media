# 前端用户管理界面更新

## 功能概述

用户管理界面新增了以下功能：
1. 显示用户最近登录时间
2. 显示用户启用/禁用状态
3. 快速启用/禁用用户按钮
4. 编辑用户时可修改启用状态

## 界面变更

### 用户列表表格

新增两列：

1. **状态列**
   - 显示用户当前启用状态
   - 启用：绿色标签 "启用"
   - 禁用：灰色标签 "禁用"

2. **最近登录列**
   - 显示用户最后一次登录时间
   - 如果从未登录，显示 "从未登录"
   - 时间格式：本地化时间格式（如：2026/3/29 22:30:45）

### 操作按钮

每个用户行的操作按钮更新为：

1. **编辑按钮**（保持不变）
   - 打开编辑对话框

2. **启用/禁用按钮**（新增）
   - 如果用户已启用：显示黄色 "禁用" 按钮
   - 如果用户已禁用：显示绿色 "启用" 按钮
   - admin 账户的禁用按钮被禁用

3. **删除按钮**（保持不变）
   - 删除用户
   - admin 账户的删除按钮被禁用

### 编辑用户对话框

新增字段：

**启用状态开关**
- 仅在编辑模式下显示
- 使用 Element Plus 的 Switch 组件
- 显示 "启用" / "禁用" 文本
- admin 账户的开关被禁用

## 用户交互流程

### 1. 查看用户列表

用户列表显示所有用户的完整信息：
- 用户名
- 角色（管理员/普通用户）
- 状态（启用/禁用）
- 创建时间
- 最近登录时间

### 2. 快速禁用用户

1. 点击用户行的 "禁用" 按钮
2. 弹出确认对话框：
   ```
   确定要禁用用户 xxx 吗？禁用后该用户将无法登录。
   ```
3. 点击 "确定" 后：
   - 调用 API 禁用用户
   - 显示成功提示
   - 刷新用户列表
   - 按钮变为绿色 "启用" 按钮

### 3. 快速启用用户

1. 点击用户行的 "启用" 按钮
2. 直接调用 API（无需确认）
3. 显示成功提示
4. 刷新用户列表
5. 按钮变为黄色 "禁用" 按钮

### 4. 编辑用户时修改启用状态

1. 点击 "编辑" 按钮
2. 在对话框中看到 "启用状态" 开关
3. 切换开关状态
4. 可以同时修改其他字段（用户名、密码、角色）
5. 点击 "确定" 保存所有修改

## API 调用

### 新增 API 函数

```typescript
// 启用用户
export async function enableUser(username: string): Promise<UserInfo>

// 禁用用户
export async function disableUser(username: string): Promise<UserInfo>
```

### 更新的接口类型

```typescript
interface UserInfo {
    username: string
    role: string
    created_at: string
    last_login?: string      // 新增
    enabled: boolean         // 新增
}

interface UpdateUserRequest {
    new_username?: string
    password?: string
    role?: string
    enabled?: boolean        // 新增
}
```

## 权限控制

1. **查看用户列表**：需要管理员权限
2. **启用/禁用用户**：需要管理员权限
3. **编辑用户启用状态**：需要管理员权限
4. **特殊限制**：
   - admin 账户不能被禁用
   - admin 账户不能被删除

## 视觉设计

### 状态标签颜色

- 启用状态：`el-tag type="success"`（绿色）
- 禁用状态：`el-tag type="info"`（灰色）

### 按钮颜色

- 启用按钮：`type="success"`（绿色）
- 禁用按钮：`type="warning"`（黄色）
- 编辑按钮：默认（蓝色）
- 删除按钮：`type="danger"`（红色）

### 表格列宽

- 用户名：150px
- 角色：100px
- 状态：100px
- 创建时间：180px
- 最近登录：180px
- 操作：280px（增加以容纳新按钮）

## 错误处理

### 禁用用户失败

可能原因：
- 尝试禁用 admin 账户
- 网络错误
- 权限不足

错误提示：显示后端返回的错误信息

### 启用用户失败

可能原因：
- 用户不存在
- 网络错误
- 权限不足

错误提示：显示后端返回的错误信息
